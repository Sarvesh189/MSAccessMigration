Welcome to MS Access Migration Tool

Simpler way to migrate MS Access Application.

Migration

Click on the Migration main menu item.

 

Below Migration form will prompt up.

1.   Select Source MS Access DB file needs to be migrated.

2.   Select Destination MS Access DB file if file already exists. You can type the path and file name in Destination text box as well.

3.   Click Analyze button to analyze the source MS Access file.

4.   Analysis will fill up the left side text box and grid as shown below.

 

5.   Now click on the Migration button which will migrate the source MS Access DB to Destination MS Access DB. Background of the migrated Item will become green in the grid and IsMigrated column will be checked out as shown below.

6.   After migration Destination left side pane controls display the migrated items summary and itemized details in the below grid.

Migration with MS SQL Server

 

1.   If it is required to migrate the local tables to sql server, check the Migrate to Sql checkbox.

It will prompt the dialog box with list of local tables. Select the tables to migrate to sql server and click on OK button.

Note: Sqlconnection string is in the appsetting file.

7.   Now if you click the migrate button. Items will be migrated to destination access file and local selected tables will be migrated to sql server and tables will be linked to destination msaccess db.
